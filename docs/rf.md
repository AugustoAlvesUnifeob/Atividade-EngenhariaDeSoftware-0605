# Regras de Negócio - Rede de cinema

## 01 - Cadastro de Cinema
O Sistema deve permitir o cadastro da filial do cinema contendo: CNPJ, localização, capacidade, numero de salas, etc.

## 02 - Cadastro de Filme
O Sistema deve permitir que o funcionário/gerenciador do sistema cadastre filmes, contendo: Titulo, classificação indicativa e tempo de exibição.

## 03 - Cadastro de Sessões
O Sistema deve permitir que o funcionário/gerenciador do sistema cadastre sessões, informando a sala, capacidade, filme e horários.

## 04 - Cadastro de Usuário
Se quiser, o usuário tem a possibilidade de cadastrar uma conta, para participar de promoções, ter recomendações de preferência e receber notificações.

## 05 - Consulta de programação
O usuário, logado ou não, poderá consultar a programação em até 2 semanas de antecedencia.

## 06 - Consulta de público
O usuário, logado ou nçao, poderá consultar as cadeiras reservadas para a sessão que desejar.

## 07 - Reserva de assento
O usuário poderá reservar sua cadeira na sessão que desejar.

## 08 - Cadastro de promoção
O funcionário/gerenciador do sistema poderá cadastrar promoções e combos e decidir se irá enviar notificações.
