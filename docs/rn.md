# Regras de Negócio - Rede de Cinema

## RN01 - Limite de publico 
O sistema só deve permitir reservar assento caso ainda não esteja com a sala lotada.

## RN02 - Horário das sessões
O sistema deve respeitar os horários cadastrados e não sobrepor uma sessão em uma sala já ocupada.

## RN03 - Dados obrigatórios
A filial da rede só pode ser cadastrada com a devida autorização e não pode faltar nenhum dado obrigatório (como CNPJ)

## RN04 - Maioridade 
Um usuário/cliente menor de idade não poderá finalizar compras nem visualizar filmes com faixa etária maior que sua idade.
