# Requisitos Não Funcionais - Rede de Cinema

## RNF01 - Facilidade de uso
O sistema deve ser de facil utilização para todos os usuários.

## RNF02 - Desempenho
O sistema não pode travar no carregamento.

## RNF03 - Acessibilidade
O sistema deve ser compativel com leitores de tela e ter opção de alto contraste.

## RNF04 - Segurança
O sistema deve manter segurança e privacidade em todas as operações, em especial as bancárias.

## RNF05 - Manutenção
O sistema deve ser organizado de forma que futuras manutenções sejam fáceis.
